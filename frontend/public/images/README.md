# SlabDash Logo Assets

Add these 3 logo files to this directory:

## Required Files:

1. **logo-full.png** - Full "SLABDASH" logo with text
   - Used on: Login page, Register page
   - Recommended size: 400x100px (or similar aspect ratio)

2. **logo-icon.png** - Square "S" icon logo
   - Used on: Sidebar (top left corner of dashboard)
   - Recommended size: 128x128px or 256x256px

3. **logo-icon-alt.png** - Alternative/angled "S" icon (optional)
   - For future use or special placements
   - Recommended size: 128x128px or 256x256px

## Where Logos Appear:

- **Login Page**: Full logo centered at top
- **Register Page**: Full logo at top left
- **Dashboard Sidebar**: S icon next to "SlabDash" text
- **Mobile Menu**: S icon when sidebar is collapsed

## File Format:
- PNG format with transparent background preferred
- SVG also acceptable for scalability
